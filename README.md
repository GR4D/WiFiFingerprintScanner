# WiFiFingerprintScanner
<b>Projekt dyplomowy inżynierski:</b> Środowisko akwizycji danych dla potrzeb lokalizowania w oparciu o sygnatury radiowe WiFi.

-Aplikacja na system Android</br>
-Struktura bazy danych SQL
